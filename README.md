# ui-ux-third-project
My third practice project for HTML and CSS

## Colour Scheme
This project uses a beautiful pastel colour palette featuring:
- **Primary Colour**: RGB(154, 127, 174) - Soft Pastel Purple
- **Secondary Colours**: Pastel Lavender, Soft Pink, and Cream tones
- **Background**: Light lavender (#faf8fc) for a soft, welcoming feel

These colours create a calming and elegant aesthetic perfect for a florist website.

## Purpose
This website serves as the homepage for **Bloom & Blossom**, a premium florist business. The purpose includes:
- **Showcase Products**: Display featured flower arrangements with descriptions and pricing
- **Promote Services**: Highlight key offerings such as custom arrangements, same-day delivery, gift wrapping, and subscription plans
- **Build Customer Engagement**: Provide contact information, newsletter signup, and social media links
- **Establish Brand Identity**: Present a professional and elegant brand through visual design and compelling content
- **Drive Sales**: Encourage visitors to shop and explore the business's offerings

The website is designed to be user-friendly, visually appealing, and responsive across all devices.
